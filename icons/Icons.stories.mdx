# Mrc Icon Components

There are 2 kinds of icons:
- Raw svg icons to use as a src on images via webpack magic
- Flexible React componenet icons

## Flexible React Component Icons

The more flexible React component icons allow to change colors and sizes via prop.

## Usage

There are some enums to get lists of possible values for color, size and stroke-width property.

```jsx
import { COLOR, SIZE, STROKE_WIDTH } from '../icons/index';

<AnyIcon color={COLOR.SUCCESS} size={SIZE.XSMALL} strokeWidth={STROKE_WIDTH.MIDI} />
```

There are special values for color and size which make the icons adaptive to their environment:

- current-color: Use current text color
- inline: Adapt to text size

```jsx
<AnyIcon color={'current-color'} size={'inline'} />
```

## Create New Component Icons

As seen further below, import `iconClassNames` and `iconPropTypes`. Set the proptypes accordingly.  

Then, use `iconClassNames` to generate some class names we will need. It generates the following variables

```jsx
{   circleClassName,
    svgClassName,
    colorToStrokeClassName,
    colorToFillClassName,
    widthToStrokeClassName,
} = iconClassNames(this.props);
```

- circleClassName (optional)
  Put on wrapper element to create a circled icon. Example: BusinessIcon
- svgClassName  
  Put on svg tag
- colorToStrokeClassName 
  Put on appropriate tag of your svg file where you want to translate the color property to the __stroke__ color
- colorToFillClassName
  Put on appropriate tag of your svg file where you want to translate the color property to the __fill__ color
- widthToStrokeClassName
  Put on appropriate tag of your svg file where you want to apply the __stroke-width__ property

ℹ️  Remove all inline fill, stroke, stroke-width attributes where you want to use the above functionality!
Remove attributes which are invalid in react e.g. `xml:something`.  
If your Icon does not behave as you wish, take a look if there are still inline styles clashing or other style definitions!
